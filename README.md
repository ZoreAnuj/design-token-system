# Design Token System

A three-tier design token system that bridges design and development, enabling consistent and customizable web components. It explores scalable theming and component architecture by structuring tokens from abstract foundations to concrete component properties.

## Key Features
- Three-tier token structure (global, semantic, component) for flexible theming
- Headless UI integration for unstyled, accessible component logic
- Consistent design properties across all web components
- Customizable themes with token-based overrides

## Tech Stack
- TypeScript
- React
- Tailwind CSS
- Headless UI

## Getting Started
```bash
npm install
npm run dev
```
